議價規則

行數計算 Effective Line of Code (ELOC)

  • 不含註解、不含美觀換行、不含簡單全域變數宣告行、不含引入和定義
  • 只有含有功能的程式和定義的副函式才會納入計算
  • 50行為1單位並且會根據情況做定價，並且保證你不會有任何虧損請放心
  • (例如第一項功能70行第二項功能98行，會以功能一50行和功能二100行定價)
  • 如果呼叫到一次副函式，那麼該行呼叫函式行會算為被呼叫函式的行數
  • (簡單來說有5行程式，但其中一行是呼叫7行的函式，那我會算為4+7=11行，估算為10行)
  • 根據功能困難度定價1單位，分為1000、1500、2000，困難度可以進行討論而定義

指令使用說明

  • 指令前面一定要加半形驚嘆號
  • 說明中有畫底線的皆為可以自行調整的數值或參數

可使用指令

  • !dc - 機器人會輸出Zen Studio的Discord群組連結
  • !yt - 機器人會輸出我的YouTube頻道連結
  • !osu - 機器人會輸出我的osu!個人檔案連結

賭博指令

  • !賭 number(整數值) - 使用你持有貢丸中的number顆進行賭博，CD時間為1分鐘
  • !賭 percentage(可小數但建議整數)% - 使用你持有貢丸中的percentage%進行賭博，CD時間為1分鐘
  • !賭 all - 使用你所有的貢丸進行賭博，CD時間為1分鐘
  • !給 使用者名稱(不需要加@也不要使用暱稱) number(整數值) - 給予使用者名稱之用戶number顆貢丸
  • !查 - 查詢你的貢丸數量
  • !領取 - 可以手動領取20顆貢丸，CD時間為30分鐘
